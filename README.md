### Running the project

Clone this repository :

```console

git clone https://github.com/jrlherm/deliveroo-frontend--LeReacteur.git
cd deliveroo-frontend--LeReacteur

```

Install packages :

```console
yarn install
```

When installation is complete, run the project :

```console
yarn dev
```
